This project contains SQL practice queries and analysis performed on a Property24 real estate dataset. The goal was to explore property data across South Africa using filtering, sorting, and aggregate functions to extract meaningful market insights.

## Project Objectives

Query and display property data

Filter properties by province, price, bedrooms, and parking

Sort properties by price, repayment, and floor size

Use aggregate functions to analyze pricing trends

Identify high-value and affordable regions

Extract insights on income requirements and property distribution
